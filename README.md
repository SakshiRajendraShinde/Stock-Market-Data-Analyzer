# Stock Market Data Analyzer

## Overview

A Python-based financial analytics project that fetches stock market data and performs trend analysis, return analysis, volatility measurement, and visualization.

---

## Problem Statement

Investors and analysts need tools to analyze stock price movements, measure risks, and understand market trends.

This project automates:
- Data collection
- Financial analysis
- Visualization
- Reporting

---

## Features

- Real-time stock data fetching
- CSV export
- Daily returns analysis
- Moving averages
- Volatility analysis
- Trend visualization
- Financial reporting
- Optional Streamlit dashboard

---

## Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- yfinance
- Streamlit
